## Tutorial ##
Play: Starts up the siren.
Stop: Kills it off.

Rotor Type: Changes to your selection of siren (sound waves).
Base Pitch: Defines the sirens current volume + frequency.
Drop Time: When stop is met, it will take (value of this) long
before fully stopping.
