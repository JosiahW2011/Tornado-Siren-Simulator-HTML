## Tutorial ##
Play: Starts up the siren.</br>
Stop: Kills it off.</br>
</br>
Rotor Type: Changes to your selection of siren (sound waves).</br>
Base Pitch: Defines the sirens current volume + frequency.</br>
Drop Time: When stop is met, it will take (value of this) long</br>
before fully stopping.
